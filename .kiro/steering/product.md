# Product Overview

九州大学工学部電気情報工学科の卒業論文「誤分類速度に着目した PGD 攻撃のロバスト性評価の再検証」の実験基盤プロジェクト．Madry らの PGD 攻撃の検証を拡張し，損失収束だけでなく**誤分類達成反復数**に着目して PGD 攻撃の誤分類特性を評価する．さらに，DeepFool の境界情報を用いた新たな初期化手法（DeepFool 初期化・Multi-DeepFool 初期化）を提案し，その効果を検証する．

## Core Capabilities

- **PGD 攻撃の実行と可視化**: 複数リスタートによる PGD 攻撃のロス曲線・正誤ヒートマップ・画像パネルを生成
- **DeepFool 初期化・Multi-DeepFool 初期化の評価**: DeepFool の境界情報を PGD の開始点に活用し，ランダム初期化との誤分類達成反復数を比較
- **誤分類性能の定量評価**: 誤分類達成率，誤分類達成反復数（平均・中央値・P95）による統計的評価
- **クロスモデル比較分析**: 自然訓練・敵対訓練など複数モデル間で共通のテストサンプルを用いた公平な比較
- **実行時間・誤分類率の統計分析**: 初期化手法ごとの実行時間計測と誤分類率のヒートマップ・LaTeX テーブル生成

## Research Context

### 論文構成

卒業論文は以下の構成で `docs/卒論/` に LaTeX ソースとして管理される:

1. **序論**: 研究背景と目的
2. **関連研究**: PGD 攻撃と DeepFool の理論的基盤
3. **評価方法**: 誤分類達成反復数・誤分類達成率の定義
4. **提案手法**: DeepFool 初期化と Multi-DeepFool 初期化
5. **評価実験**: MNIST/CIFAR-10 での実験結果
6. **結論**: 知見のまとめと今後の展望
- 付録: 計算コスト分析，ロス曲線の追加図

### 提案手法

- **DeepFool 初期化**: DeepFool で最も近い決定境界方向の敵対的サンプルを生成し，L-inf 制約に射影して PGD の初期点とする
- **Multi-DeepFool 初期化**: 正解ラベル以外の全ターゲットラベルに対して個別に DeepFool を実行し，(K-1) 個の多様な初期点を生成

### 評価指標

- **誤分類達成反復数** (t*): モデルの予測が正解から不正解に変わった最初の反復数
- **誤分類達成率** (R_m): 全試行のうち最終反復時点で誤分類している割合
- **統計量**: 平均，中央値，95 パーセンタイル

## Target Use Cases

- MNIST / CIFAR-10 データセットに対する敵対的攻撃の初期化手法の比較実験
- 複数モデル（nat, adv, nat_and_adv, weak_adv）に対する攻撃評価
- 卒業論文で使用する図表・統計データの生成
- Singularity コンテナ上での大規模バッチ実験の実行

## Value Proposition

従来のランダム初期化による PGD 攻撃に加え，DeepFool ベースの初期化が攻撃性能に与える影響を体系的に評価する．損失の収束だけでなく，誤分類がいつ達成されるか（誤分類達成反復数）という新たな観点から PGD 攻撃を再検証し，初期化手法の選択がロバスト性評価の結論に影響するかを明らかにする．
